# TradingAgents — AI Multi-Agent Trading Analysis Platform

منصة التحليل الذكي بالذكاء الاصطناعي للمتداولين

## 🚀 المميزات

- **تحليل متعدد الوكلاء** — فريق من 5 محللين ذكاء اصطناعي يعملون معاً
- **اختيار الوكلاء** — اختر أي محلل تريد: فني، أساسي، معنويات، أخبار، مخاطر
- **رسوم بيانية احترافية** — مدمج مع TradingView للتحليل المباشر
- **دعم متعدد المزودين** — DeepSeek, Groq, Google Gemini, OpenAI, Ollama
- **تصميم احترافي** — واجهة داكنة مناسبة لبيئة المتداولين

## 📁 الملفات

| الملف | الوصف |
|-------|-------|
| `server.py` | الخادم الرئيسي (FastAPI) |
| `templates/index.html` | الواجهة الأمامية |
| `requirements.txt` | المكتبات المطلوبة |
| `.env` | إعدادات API Keys |
| `run.bat` | ملف التشغيل |

## 🔧 التثبيت والتشغيل

1. ضع مفتاح API في ملف `.env`:
```
DEEPSEEK_API_KEY=your_key_here
# أو أي مزود آخر
```

2. شغّل ملف `run.bat`

3. افتح المتصفح على `http://localhost:8000`

## 🎯 اختيار الوكلاء

يمكنك اختيار أي مجموعة من الوكلاء التالية:
- 🤖 **فريق كامل** — جميع المحللين
- 📊 **المحلل الفني** — التحليل الفني والمؤشرات
- 💬 **محلل المعنويات** — تحليل المشاعر والاتجاهات
- 📰 **محلل الأخبار** — الأخبار الاقتصادية
- 📋 **المحلل الأساسي** — البيانات المالية
- 🛡️ **محلل المخاطر** — إدارة المخاطر
- ⚔️ **مناقشة Bull vs Bear** — مناظرة الاتجاهات
